# GameLink - 동호회 전용 게이밍 VPN 설치 안내

안녕하세요! 동호회 멤버끼리 **온라인에서도 LAN 게임**을 즐길 수 있도록
전용 VPN 프로그램 **GameLink**를 준비했습니다.

한 번 설치하면 더블클릭 한 번으로 자동 연결됩니다!

---

## 어떤 프로그램인가요?

- 동호회 멤버끼리만 연결되는 **전용 VPN**입니다
- 설치하면 `GameLink.exe` 하나로 연결/해제가 됩니다
- 시스템 트레이(화면 오른쪽 아래)에 상주하며, 연결 상태를 아이콘 색상으로 알려줍니다
  - ⚪ 회색: 대기 중
  - 🟡 노랑: 연결 시도 중
  - 🟢 초록: 연결 성공!
  - 🔴 빨강: 연결 끊김

---

## 설치 방법 (처음 한 번만!)

### STEP 1. 파일 다운로드

아래 링크에 접속합니다:

👉 **https://github.com/uygnoey/n2n-gaming-lan**

<!-- 📸 스크린샷: GitHub 메인 페이지 (ss_9886ujn0k) -->

1. 초록색 **`<> Code`** 버튼을 클릭합니다
2. 드롭다운 메뉴 맨 아래 **`Download ZIP`** 을 클릭합니다

<!-- 📸 스크린샷: Code 드롭다운 → Download ZIP (ss_2797barxf) -->

3. `n2n-gaming-lan-main.zip` 파일이 다운로드됩니다

---

### STEP 2. 압축 풀기

1. 다운로드된 zip 파일을 **우클릭** → **압축 풀기** (또는 **여기에 풀기**)
2. `n2n-gaming-lan-main` 폴더가 생깁니다
3. 폴더 안에 아래 파일들이 있는지 확인하세요:

```
n2n-gaming-lan-main/
├── install_edge.ps1      ← 설치 스크립트 (이걸 실행합니다)
├── env.ini.example       ← 설정 파일 예제
├── README.md
└── ...
```

---

### STEP 3. 설정 파일 만들기

1. 폴더 안의 `env.ini.example` 파일을 **복사**합니다
2. 복사한 파일 이름을 `env.ini` 로 변경합니다
3. `env.ini`를 **메모장**으로 열어서 아래와 같이 수정합니다:

```ini
[Settings]
SuperNode=관리자가 알려준 서버 주소
Community=관리자가 알려준 커뮤니티 이름
EncryptKey=관리자가 알려준 암호키
```

> ⚠️ **SuperNode, Community, EncryptKey 값은 동호회 관리자에게 문의하세요!**
> 모든 멤버가 같은 값을 사용해야 서로 연결됩니다.

---

### STEP 4. PowerShell 열기 + 실행 정책 설정 + 스크립트 실행

Windows는 기본적으로 PowerShell 스크립트(.ps1) 실행을 차단합니다.
아래 순서대로 따라하세요!

#### 4-1. PowerShell을 관리자 권한으로 열기

1. 키보드에서 **Windows 키**를 누르고 `PowerShell` 을 입력합니다
2. 검색 결과에서 **Windows PowerShell** 을 찾아 **우클릭** → **관리자 권한으로 실행** 을 선택합니다
3. "이 앱이 디바이스를 변경할 수 있도록 허용하시겠어요?" 팝업이 나오면 **예** 를 클릭합니다
4. 파란색 PowerShell 창이 열립니다!

#### 4-2. 실행 정책 설정

**아래 두 가지 방법 중 하나를 선택하세요.**

**✅ 방법 1: 영구 설정 (권장 - 한 번만 하면 끝!)**

이 설정을 하면 앞으로 PC에서 모든 PowerShell 스크립트를 자유롭게 실행할 수 있습니다.

1. 아래 명령어를 **복사 & 붙여넣기** 후 Enter를 누릅니다:

```
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
```

2. 아래와 같은 확인 메시지가 나옵니다:

```
실행 규칙 변경
실행 정책은 신뢰하지 않는 스크립트로부터 사용자를 보호합니다. ...
[Y] 예(Y)  [A] 모두 예(A)  [N] 아니요(N)  [L] 모두 아니요(L)  [S] 일시 중단(S)  [?] 도움말 (기본값은 "N"):
```

3. `Y` 를 입력하고 Enter를 누릅니다.

4. 에러 메시지 없이 다음 줄로 넘어가면 성공입니다!

> 💡 이 설정은 **한 번만 하면 됩니다.** 이후부터는 .ps1 스크립트를 자유롭게 실행할 수 있습니다.

**방법 2: 일회성 실행 (PC 설정 변경 없이)**

PC 설정을 바꾸고 싶지 않다면 4-2를 건너뛰고, 아래 4-3에서 스크립트 실행할 때 다른 명령어를 사용하면 됩니다. (아래 4-3 참고)

#### 4-3. 설치 스크립트 실행

1. PowerShell에서 압축 푼 폴더로 이동합니다:

```
cd "압축 푼 폴더 경로"
```

예시:
```
cd "C:\Users\내이름\Downloads\n2n-gaming-lan-main\n2n-gaming-lan-main"
```

> 💡 **팁:** 파일 탐색기에서 폴더 상단의 주소 표시줄을 클릭하면 경로를 복사할 수 있습니다!

2. 스크립트를 실행합니다:

**위에서 방법 1(영구 설정)을 했다면:**
```
.\install_edge.ps1
```

**방법 2(일회성)를 선택했다면:**
```
powershell -ExecutionPolicy Bypass -File .\install_edge.ps1
```

> ⚠️ `스크립트를 실행할 수 없습니다` 또는 `보안 오류` 메시지가 나오면
> 위의 4-2로 돌아가서 실행 정책을 설정해주세요!

3. **회원 고유번호**를 입력하라는 메시지가 나옵니다
   - 관리자에게 받은 본인의 회원번호를 입력하세요
   - 입력하면 VPN IP가 자동으로 계산되어 표시됩니다
   - 맞으면 Enter, 다시 입력하려면 N을 입력합니다

4. 설치가 완료되면 `C:\n2n` 폴더에 파일들이 생성됩니다!

## 사용법 (설치 후 매번)

### 연결하기

`C:\n2n` 폴더의 **`GameLink.exe`** 를 더블클릭하면 끝!

> ⚠️ 처음 실행 시 "관리자 권한으로 실행하세요" 메시지가 나오면
> GameLink.exe를 **우클릭** → **관리자 권한으로 실행** 하세요.

### 연결 확인

- 화면 오른쪽 아래 시스템 트레이에 **동그란 아이콘**이 나타납니다
- 🟢 **초록색**이면 연결 성공!
- 아이콘을 **좌클릭**하면 실시간 로그 창이 열립니다
- 아이콘을 **우클릭**하면 메뉴가 나옵니다:
  - 연결 / 재연결 / 연결 해제
  - 로그 보기
  - config.ini 열기
  - 종료

### 종료하기

트레이 아이콘 **우클릭** → **종료 (Exit)** 를 선택하세요.

## 자주 묻는 질문

**Q. GameLink.exe가 실행이 안 돼요**
→ **우클릭 → 관리자 권한으로 실행** 해보세요.

**Q. 트레이 아이콘이 빨간색이에요**
→ 인터넷 연결을 확인하고, 서버가 정상 가동 중인지 관리자에게 문의하세요.

**Q. 상대방 PC가 안 보여요**
→ 양쪽 모두 **같은 Community, Key** 값을 사용하는지 확인하세요.

**Q. 회원번호를 잘못 입력했어요**
→ `C:\n2n\config.ini` 파일을 메모장으로 열어서 `MemberID=` 값을 수정하세요.

**Q. VPN 연결은 됐는데 게임에서 안 보여요**
→ 게임 내 LAN/로컬 네트워크 검색 기능을 사용하세요.
→ 방화벽이 게임을 차단하고 있을 수 있습니다. 게임 실행 시 방화벽 허용 팝업이 나오면 "허용"을 클릭하세요.
